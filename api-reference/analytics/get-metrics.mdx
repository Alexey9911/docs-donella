---
title: "Get Metrics"
api: "GET https://api.bookie.app/v1/analytics/metrics"
description: "Retrieve usage metrics and transaction statistics"
---

### Authorization

<ParamField header="Authorization" type="string" required>
  Bearer token. Format: `Bearer bookie_live_abc123def456`
</ParamField>

### Query Parameters

<ParamField query="startDate" type="string" required>
  Start date in ISO 8601 format
</ParamField>

<ParamField query="endDate" type="string" required>
  End date in ISO 8601 format
</ParamField>

<ParamField query="granularity" type="string" default="day">
  Data granularity. Options: `hour`, `day`, `week`, `month`
</ParamField>

### Response

<ResponseField name="period" type="object">
  Time period for metrics
</ResponseField>

<ResponseField name="transactions" type="object">
  Transaction statistics
  
  Properties:
  - `total` (number): Total transactions
  - `swaps` (number): Token swap transactions
  - `transfers` (number): Transfer transactions
  - `successRate` (number): Success rate percentage
</ResponseField>

<ResponseField name="volume" type="object">
  Trading volume statistics
  
  Properties:
  - `totalUSD` (number): Total volume in USD
  - `averagePerTransaction` (number): Average transaction size
</ResponseField>

<ResponseField name="intents" type="object">
  Intent parsing statistics
  
  Properties:
  - `total` (number): Total intents parsed
  - `averageConfidence` (number): Average confidence score
  - `parseErrors` (number): Failed parses
</ResponseField>

<RequestExample>
```bash cURL

  --url 'https://api.bookie.app/v1/analytics/metrics?startDate=2024-03-01T00:00:00Z&endDate=2024-03-08T23:59:59Z' \
  --header 'Authorization: Bearer bookie_live_abc123def456'
```
